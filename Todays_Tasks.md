# Today's Tasks

> Jot down what you need to get done, one line per task. Add a date header for each day if you want to keep a running history. The scheduled assistant reads this file each morning and sends you a digest of what's listed under today's date (or under "Today's Tasks" if no date headers are used).

## 2026-06-06

- [ ] Example: Reply to client email about COBOL migration timeline
- [ ] Example: Review chapter 4 of COBOL study plan
- [ ] Example: Call dentist to reschedule appointment

---

Tips:
- Use `- [ ]` for open tasks and `- [x]` for done ones.
- Add a new `## YYYY-MM-DD` section each day, or just keep editing the list under one running header — either works.
